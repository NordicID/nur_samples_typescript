# NurApi API Docs

Local copies of the `@nordicid/nurapi`, `@nordicid/nurapi-web`, and `@nordicid/nurapi-node` API documentation (`.api.md`) for offline use and AI assistant integration.

The packages themselves are installed from npm — see `package.json` in each sample.

**Online docs:** https://nordicid.github.io/nur_nurapi_typescript/

## Updating API Docs

Run the update prompt in VS Code Copilot Chat:

```
#prompt:update-nurapi
```

Or update manually — see [.github/prompts/update-nurapi.prompt.md](../.github/prompts/update-nurapi.prompt.md) for the full steps.
