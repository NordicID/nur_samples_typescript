# NurApi Packages & API Docs

This folder contains the `@nordicid/nurapi`, `@nordicid/nurapi-web`, and `@nordicid/nurapi-node` npm packages (`.tgz`) and their API documentation (`.api.md`).

**Source:** https://nordicid.github.io/nur_nurapi_typescript/

## Updating

Run the update prompt in VS Code Copilot Chat:

```
#prompt:update-nurapi
```

Or update manually — see [.github/prompts/update-nurapi.prompt.md](../.github/prompts/update-nurapi.prompt.md) for the full steps.
